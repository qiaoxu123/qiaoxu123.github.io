分割线 -- 2019年8月8日更新

最近又重新安装了Linux mint系统，从这两周的使用感受来看，比Ubuntu更加简洁和便利。回到正题，又需要重新安装一次ROS。经过前两次安装，经验丰富多了，在这重新整理下。

第一，查看官网安装教程。其中存在问题，apt update检索不到，意料之中。

第二，查找相应命令，找到该地址，按照步骤轻松解决。之后回到官网教程，完成后面的步骤。

命令如下：

sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu bionic main" > /etc/apt/sources.list.d/ros-latest.list'
