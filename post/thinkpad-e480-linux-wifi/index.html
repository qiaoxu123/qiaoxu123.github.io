ThinkPad E480 Linux WIFI 问题解决

发布于 2019-09-03 | 分类于 工程实现 | 1分钟 | 205字数

因为工作需要在个人笔记本上安装Linux，目前使用的版本是Linux mint19.2。但ThinkPad E480 因为自带的8821ce无线网卡太“新”，导致无法直接识别，测试过网上各种方案后，终于找到一个比较稳定的方案。在这笔记记录下。

下载该文件，具体来源已忘记，是某个github仓库提供的。下载链接

下载完成后，放置在任意目录下解压。然后执行下面代码即可：

make
sudo make install
sudo modprobe -a 8821ce

注意：

  1. 目前仅在Ubuntu16.04， Ubuntu18.04，以及Linux mint19版本上测试没有问题，网络上有各种说内核存在问题，暂时还没有遇到。
  2. 目录尽可能避开工作环境，因为一旦删除wifi又会识别不到。
