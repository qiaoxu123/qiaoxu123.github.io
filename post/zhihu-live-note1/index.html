一、德国和英国博士全奖申请过程

1. 前期准备

留学分为两种：一种为全奖申请，另一种为国家留学资助CSC

1.1. 注意留意国外博士职位的发布：

  • 国外网站：Research Gate https://www.researchgate.net/home
  • 英国博士职位网站：http://www.jobs.ac.uk/phd
  • 欧盟博士职位：https://euraxess.ec.europa.eu/
  • 玛丽居里项目网址：http://www.jobs.ac.uk/phd
  • 德国DAAD网址：https://www.daad.de/en

1.2. 从小木虫，或者丁香花园上获知

1.3. 从研究生期间的项目中联系获知

1.4. 参加国家会议，与大牛教授聊

1.5. 通过在研究生期间的项目认识相关的导师，获得认可，这样几率更大。

2. 如何跟教授套词

注意事项：

  • 不要发模板（教授很忙，申请信必须非常有亮点）
  • 知道教授focus在哪一点
  • 知道自己背景中的亮点
  • 让自己的背景与教授切合

面试过程注意事项：

  • 准备好问教授的问题
  • 研究下教授的paper
  • 注意你自己发表的文章
  • 英语口语交流不用特别担心，保证基本沟通即可
  • 面试时可同学帮忙打辅助，在一旁帮你听
  • 看清学费和生活费

二、欧盟玛丽居里ITN项目简介和申请流程

玛丽居里项目网址：http://www.jobs.ac.uk/phd

包含生活费和研究经费，主要面向国际生，同时各个大学分配博士名额。

三、国家公派留学申请流程

1. 基本情况

根据2017年，共9500个联陪名额，3500个攻博名额

  • 对于攻博名额，分为两种，一种是国家选取进行高水平合作，成功率很高；另一种是自己申请，成功率相对低

  • 对于联陪名额，每个学校都有自己的名额，竞争相对较小

总体来说，不一定要paper和语言成绩，只要外导能够开语言证明即可

2. 申请流程

先拿offer(和外导申请)

3，4月份提交材料，先通过学院审核，后递交国家留学基金委

材料包括：

  • 外导邀请信
  • 外导简历
  • 研修计划（重要，中英文两份。先写长，后修裁）
  • 学历证明，成绩单，硕士导师简历

注意事项：

  • 小心广发邀请信的导师
  • 本科学历不是那么重要，重要的是科研成绩，paper

三、硕士期间科研

  • 多跟导师沟通，交流想法
  • 做好基础项目，不要好高鹜远，稳扎稳打

发论文：

  • 找前沿方向（发文章比较多）
  • 从最基础的文章开始读（有自己的架构和概念）
  • 复现别人的成果，有自己的想法，弥补别人不足
  • 平时注意积累小论文，慢慢搭积木
  • 从写，审，发，整个周期很长，注意时间问题
