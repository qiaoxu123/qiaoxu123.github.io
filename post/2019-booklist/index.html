我的2019年书单

· 2019-12-26 · # 个人感悟 # 日常生活

研究生的第一年，和本科的时光有彻彻底底的不同。经历了好多，虽然无奈，还是被生活抽走了几分稚气。竭尽全力，总算是读了几本闲书，认识了一些有趣的人，做了一点踏实事。

和去年比起来数量上已经差很多了，质量上也不敢说上浮多少。向社会走出一小步，在外呆了差不多整一年后，才对自己的稚嫩和单纯有了更进一步的认识。不安和躁动成了2019年的代表词。直到最后一个月，重新拿起kindle，放下那股浮躁重新开始读，才终于感觉到自己又往前迈出了一步。

1月：泽敏在东师准备出国英语备考，正好临近研一期末考试，每天的准备考试加上后面往返去找她，成了这个月最美好的日子；爷爷突然离开了，人生中第一次失去亲人，生命的脆弱第一次展示在我面前，有些想说的话再也说不出去了；

  • 《漫长的告别》
  • 《1984》
  • 《一直特立独行的猪》
  • 《国境以南，太阳以西》

2月：科二考过，被驾校的舅舅带，日子真的不太好过啊。每天只是简单地陪着爸妈，平凡的小日子也挺温暖的；开学了，重新见到泽敏，真的开心的不行，让我有点惊讶；婶婶去世了，第二次失去亲人，本来已经知道得病后时间不多了，但还是单纯地觉得这个时间会很久很久。真的很难受，真的好害怕，下一个失去的会是谁啊。

  • 《三体外传：理想之宙》

3月：平淡的一个月。总算感觉自己开始了研究生的生活，准备开始认真看下方向相关的论文，尝试做下仿真实验，做一个正常研究生该做的。这也是和泽敏在学校呆的最后一个月。

  • 《没有色彩的多崎作和他的巡礼之年》

4月：接到老师的项目安排，要去上海公司实习。兴致勃勃准备出发！收到人生的第一份offer；来到上海，找好房子，分配好工位和电脑，安排好公司的任务，开始正式的实习；实习并没有刚开始想的那么美好，内容很枯燥，和之前想做的方向没有一点关系，心情郁闷了好久。

  • 《今日简史》

5月：收到学校通知，必须返校一次。2019年在学校呆了最后7天，玩了好多吃了好多，好开心的日子～；回到公司，开始继续之前的任务，同时开始接触一点自动驾驶感知方面的内容，心情恢复了一些。

  • 《远方的鼓声》

6月：基本习惯了在公司实习的节奏，每天规律性的上下班，晚上有性质的话就学一些感兴趣的方向的内容，坚持了一段时间，陆陆续续有了一点自己的存货。

  • 《1Q84》

7月：预定的项目仍然没有签约，拖的时间越来越长，合同也重新续约。实习的日子变得越来越长，心里开始浮躁起来。激光雷达的感知和我预期的方向偏差太大，在接触过后也发现一点都不感兴趣。实习的生活真的糟透了。

  • 《人间失格》
  • 《罗生门》
  • 《沉默的大多数》

8月：一起来实习的师兄走了一个，而项目仍然是遥遥无期。研究生第一次暑假，在上班的日子里慢慢度过了。每天重复同样的节奏，感觉到自己的心境慢慢变了很多。之前的憧憬还有热血，都冷却了，剩下的大多是想赶快结束的挣扎；泽敏的CSC终于通过了，最后一个月在上海和她一起度过的，简单的生活，每天下班后一起上自习，很纯粹的享受最后的时光。好幸福。

  • 《乌合之众》

9月：送走了泽敏，两年后再见，心里终于还是变得空空的；收到项目要签约的消息了，这次实习的任务分配又换了一个，是自己争取的，想做的方向相关的。终于可以开始了，自己想做的。心里乐观地预判年前应该可以结束项目。

  • 无

10月：一起来的师兄又走了一个，这次只剩下我一个了。开始真正地开展签约的项目了，心里的焦虑正式成型了，想的只有尽快拜托这个项目，开始新的实习；自己投了人生中第一份简历，一家自己很想去的公司，过了一面，但笔试仍然是太差了，最终没有通过；项目进展得曲曲折折，最后定下来要出差到江苏盐城，开始正式的测试。

  • 《没有色彩的多琦作和他的巡礼之年》

11月：来到了江苏盐城，开始进一步的调试，慢慢熟悉了办公环境和居住环境。各种问题也慢慢出现，每天一点点开展；第二次面试，商汤，这次真的准备了好久，然而仍然是笔试太差，面试题一点思路都没有，底子真的是太差了。

  • 无

12月：项目进展不明朗，中间一波三折，推测要到明年调试；通过了360的面试，一直在等待offer，还是没有收到；每天和泽敏联系，喜怒哀乐都有，只想着早点见面。一想到2019年就这么过去了，心里就烦躁的不行，我可能也是得病了，找不到之前的动力和学习的心境，以前想做的事情统统不想去做。躁动的心情也到了顶峰，现在的我该怎么办呢？

  • 《刺杀骑士团长》
