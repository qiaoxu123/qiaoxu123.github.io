自动驾驶Planning总结3 -- Behaviour Planning

· 2019-08-09 · # 笔记 # 规划 # 自动驾驶

介绍Behavior Planning的相关概念，finite state machine的使用

在自动驾驶中，车辆经过的道路约束，大部分情况下是作为先验知识已知的，获取途径是经过高精度地图。

Behavior Planning中对于输入的要求较高，要求从环境中获取大量信息。在这可以大体分为这几类：

  1. 路网信息。通过高精度地图获取。
  2. 到达目的地的路径。通过在路网上的Mission Planning来告知。
  3. 车辆的自身信息。该信息对于同步高精度地图，实现车辆状态同步至关重要。
  4. 环境中的动态与静态障碍物信息。以此来完善行为操作。
