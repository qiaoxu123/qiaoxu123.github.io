Ubuntu 18/16 下 TP-LINK 5500无线网卡驱动问题

· 2019-11-25 · # 工程调试

今天刚花了半小时时间调了一下公司的主机，因为有线网络一直连接不上，一气之下找了一个无线网卡直接怼上试试，结果因为是Ubuntu系统还不适配，哎，只好耐心找了下驱动，费了些时间搞定了，在这简单记录下。

参考博客
驱动链接

如何识别是不是TP-LINK 5500, 两个办法：
第一是查看USB口上有小字标识，写着驱动型号；
第二是lsusb命令，看有没有1a2b的驱动参数，如果有说明是的

TP-LINK 5500的网卡驱动是8821cu。

下载完成后，按照下面步骤依次输入命令即可：

make
sudo make install
sudo modprobe 8821cu

最后输入该命令即可

sudo usb_modeswitch -KW -v 0bda -p 1a2b
